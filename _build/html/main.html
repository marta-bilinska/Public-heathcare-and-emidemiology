Main documentation¶

This module contains the main module of the project, that is composed of experiment functions and functions that make visualisations for the project.

This module contains a main function for doing experiments with the data. !!! Notice that for this program to be faster, the file that is used in it is an excerpt from the whole file.

main.create_plot(data, diseases_lst)¶

DiseaseTree, list -> str Creates a plot of the two diseases and saves it.

main.experiment_cities(data, indicator)¶

DiseaseTree, str -> str Does experiments with city names.

main.experiment_gender(data, indicator)¶

DiseaseTree, str -> str Does experiments with gender values of all diseases linked to indicator.

main.get_cities(data)¶

DiseaseTree -> list Gets a list of all cities.

main.get_diseases(data)¶

DiseaseTree -> list Gets a list of all diseases.

main.get_ethnicities(data)¶

DiseaseTree -> list Gets a list of all ethnicities.

main.get_indicator_disease(data)¶

DiseaseTree -> set() Gets a set of tuples of diseases and indicators.

main.main_experiments()¶

() -> () Main function that conducts experiments.

main.read_into_data_structure()¶

() -> DiseaseTree This function reads the information of the example csv file, forms it as a pandas DataFrame and conducts experiments of the data.

main.total_value_experiment(data, indicator)¶

DiseaseTree, str -> str Does experiments with values of all diseases.
